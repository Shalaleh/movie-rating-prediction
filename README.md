# Movie Rating Prediction

## Overview
This project analyzes movie data and builds machine learning models to predict audience ratings.

## Technologies
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Workflow
- Data cleaning
- Exploratory Data Analysis (EDA)
- Feature selection
- Model training (Linear Regression & Random Forest)
- Model evaluation and comparison

## Results
Linear Regression performed slightly better than Random Forest, indicating a relatively linear relationship between features and audience ratings.

## Key Insights
- Critic ratings strongly influence audience ratings
- Audience engagement also plays a role
- Simpler models can perform well on structured datasets

## Conclusion
This project demonstrates a complete machine learning workflow from data preprocessing to model interpretation.
